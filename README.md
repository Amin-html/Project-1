# 🏎️ TrackBooking

Сервис бронирования гоночных треков — Проект 1 из 10.

## Стек
- Python 3.13 + Django 5.2
- SQLite
- Docker

## Функционал
- 📋 Каталог треков с фотографиями
- 📄 Детальная страница трека
- 🔐 Регистрация и вход
- 🏁 Бронирование трека на дату и время
- ⚠️ Проверка что время не занято
- ❌ Отмена бронирования
- 👤 Личный кабинет с моими бронированиями

## Запуск

```bash
git clone https://github.com/Amin-html/Project-1.git
cd Project-1
docker-compose up --build
```

Открой: `http://127.0.0.1:8000`
